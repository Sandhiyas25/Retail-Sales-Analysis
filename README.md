# Retail Sales Analysis Dashboard

## Overview

This project focuses on analyzing retail sales data to uncover valuable business insights related to sales performance, customer behavior, product trends, and market segments. The analysis was performed using Excel, SQL, Python, and Power BI to transform raw sales data into meaningful visualizations and actionable insights.

## Project Objective

The primary objective of this project is to:

* Analyze sales trends and revenue performance.
* Identify top-performing products and categories.
* Understand customer purchasing behavior.
* Evaluate sales performance across different states and segments.
* Create an interactive dashboard for business decision-making.

## Tools & Technologies Used

### Microsoft Excel

* Cleaned and prepared raw retail sales data.
* Handled missing values and duplicate records.
* Standardized data formats and performed basic data validation.
* Created initial summaries and data transformations.

### SQL (MySQL)

* Imported cleaned datasets into the database.
* Performed data analysis using:

  * JOIN operations
  * Aggregate Functions
  * GROUP BY clauses
  * Subqueries
  * Window Functions
* Generated insights on:

  * Total sales
  * Product performance
  * Customer behavior
  * Revenue trends
  * Category-wise analysis

### Python

* Used Python libraries such as:

  * Pandas
  * NumPy
  * Matplotlib
  * Seaborn
* Conducted exploratory data analysis (EDA).
* Created visualizations to identify trends and patterns.
* Analyzed sales distribution, customer segments, and product performance.

### Power BI

* Built an interactive multi-page dashboard.
* Created KPIs and DAX measures.
* Designed dynamic visualizations with filters and slicers.
* Enabled drill-down analysis for detailed insights.
* Developed dashboards for:

  * Sales Analysis
  * Segment Analysis
  * Customer Analysis

## Key Findings

* April recorded the highest sales among all months.
* The product **Trek Slash 8 27.5 – 2016** generated the highest revenue.
* New customers contributed significantly to overall sales.
* New York recorded the highest sales among analyzed states.
* Trek emerged as the leading brand in terms of quantity sold.
* Most customer orders were successfully delivered.
